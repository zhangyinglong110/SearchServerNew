1.首页查询接口:
url: editPageServlet
请求的参数：
largeName 大区名称
schoolName 校区的名称
majorName 专业名称
返回的数据格式为： {"code":200, "msg":" success", "results":[ { "teacherName":"Dear Linda", "role":"讲师", "className":"web前端1班" "majorName":"web前端", "majorId":1, "schoolId":1 } ] }
--------------------------------------------------------------------------------------
2.新增数据接口:
url:editPageInsertServlet
请求参数:
schoolName 校区名称
teacherName 老师名称
className 班级名称
role 角色
majorName 专业名称
返回数据： 直接返回一个字符串："插入成功！"
---------------------------------------------------------------------
3.删除某一条数据的接口:
url : deleteEditPageServlet
请求参数:
schoolName 校区名称
teacherName 老师名称
className 班级名称
role 角色
majorName 专业名称
返回数据：
直接返回一个字符串："删除成功！"
---------------------------------------------------------------------
